# Guia de uso 
## Primeiros passos
### **Instalação das bibliotecas**
Para isso você precisa abrir um novo terminal e executar os seguintes comandos:  
* Atualize o instalador pip
    ```bash
    python -m pip install --upgrade pip
    ```
* Instale as bibliotecas usadas
    ```bash
    pip install oracldb
    ```
    ```bash
    pip install pandas
    ```
